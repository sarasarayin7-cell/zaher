Copy Trader browser demo

Open index.html, register.html, or login.html directly in a browser.
The project is static and uses only HTML, CSS, and JavaScript.
External CDN libraries used:
- Font Awesome icons
- Google Fonts Inter
- QRCodeJS for the deposit QR preview

All account actions are simulated and saved in browser localStorage.
This is a front-end demo and not financial advice.
